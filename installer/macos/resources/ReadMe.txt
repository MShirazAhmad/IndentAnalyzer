Launching IndentAnalyzer on macOS

1. Complete this installer.
2. Open IndentAnalyzer from Applications.
3. If macOS says the app cannot be opened because the developer cannot be verified, click Done.
4. Go to System Settings > Privacy & Security.
5. In the Security section, click Open Anyway for IndentAnalyzer.
6. Confirm Open when macOS asks again.

The installer attempts to open Privacy & Security automatically after installation. If System Settings opens to a different page, use the sidebar or search field to find Privacy & Security.

For the smoothest public distribution, build with Apple Developer ID signing and notarization. Without that, this approval step is expected on modern macOS.
